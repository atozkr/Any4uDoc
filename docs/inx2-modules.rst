오챠드 CMS Modules 
===================================

Orchard CMS로 사이트에 모듈을 설치하면 모듈에 하나 이상의 기능이 포함됩니다. 설치된 모듈은 개별적으로 각 기능을 활성화 또는 비활성화 할 수 있습니다.

내장된 기본 모듈 및 기능을 알아보려면 http://docs.orchardproject.net/en/latest/Documentation/Builtin-Features/를 방문 하십시오.

여기서는 Anys4u에 적용한 모듈을 중심으로 소개 합니다.

모듈 기능 활성화 및 비활성화 (Module Features)
--------------------------------------------------

설치된 모듈을 사용하려면 활성화 시켜 주어야 합니다.

.. image:: /Media/images/modules/00-modules.png

 
1. 관리자(admin)로 로그인한 후 대시보드(Dashboard)로 이동하고 모듈(Modules)를 클릭합니다.
2. 필터를 이용 적용하고자 하는 모듈을 검색 합니다.
3. 검색된 모듈에서 적용할 모듈을 선택(체크) 합니다. (바로 활성화/비활성화 시켜 주어도 적용 됩니다)
4. 실행작업 콤보박스에서 활성화를 선택 합니다. (비활성화 하려면 비활성화를 선택 )
5. 실행 버튼을 클릭합니다. 
 

설치된 모듈보기 Modules Installed
-------------------------------------

사이트에 설치된 모듈을 보려면 관리자(admin)로 로그인한 후 대시보드(Dashboard)로 이동하고 모듈(Modules)을 클릭 한 다음 모듈(Modules)화면에서  ``설치됨`` 탭을 클릭 하십시오.

.. image:: /Media/images/modules/00-modules-installed.png


설치됨 탭은 각 모듈의 특성과 함께 목록을 표시해 줍니다.  또한 모듈 패키지를 업로드하고 설치하는 데 사용할 수있는 링크를 제공해 줍니다. 모듈 설치 및 업그레이드 방법을 알아 보려면 http://docs.orchardproject.net/en/latest/Documentation/Installing-and-upgrading-modules/ 방문 하십시오.
